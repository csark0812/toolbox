# Sensitive source material

Place Slack exports, writing samples, and other identity source files here.

Files in this directory are **git-crypt encrypted** in the repository. They sync across your machines only after `git-crypt unlock`.

Do not copy contents into `personal/<slug>/SKILL.md` or `team/` skills.

## Suggested files

- `writing-samples.md` — curated excerpts (not full Slack dumps)
- `slack-export-YYYY.md` — optional raw export for local calibration only
